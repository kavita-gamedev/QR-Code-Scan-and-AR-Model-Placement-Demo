# QR-Code-Scan-and-AR-Model-Placement-Demo
how to scan a QR code and use Augmented Reality (AR) to place a 3D model in the real world using Unity and AR Foundation.

This Unity project demonstrates how to use **QR Code scanning** and **Augmented Reality (AR)** to place a 3D model in the real world using **AR Foundation** and **ZXing.Net** for QR code scanning.

## Features

- Scan a **QR code** to trigger the placement of a 3D model in AR.
- Use **AR Foundation** to place the model in the real-world environment.
- Mobile support for both **iOS** and **Android** devices.
